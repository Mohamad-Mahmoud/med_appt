# Medical Appointment Booking Website

This project is a responsive front-end web application for booking medical appointments.  
It is developed as part of a front-end development capstone and focuses on UI/UX, accessibility, and modern React practices.

---

## Overview

The website allows patients to:

- View clinic information and services
- Browse doctors and specialties
- Check available appointment dates and times
- Submit a booking request via an appointment form
- Receive basic confirmation/feedback for their booking

---

## Tech Stack

- **Framework:** React (JavaScript, ES6+)
- **Styling:** CSS (and responsive layout using flexbox/grid)
- **Routing:** React Router (single-page app navigation)
